# Shopware Account Plugin Github Action
Plugin Actions with Shopware Account.

```
-   name: Plugin Validate
    uses: webpiloten/shopware-account-plugin-github-action@v1
    with:
      command: validate
```

```
-   name: Plugin Update
    uses: webpiloten/shopware-account-plugin-github-action@v1
    with:
      command: update
```

```
-   name: Plugin Upload
    uses: webpiloten/shopware-account-plugin-github-action@v1
    with:
      command: upload
```
